1. IN ORDER TO / SO AS TO + V

  • In order not to catch coronavirus, you should stay at home.
  • You should get vaccines in order to protect you from coronavirus.
  • I go to bed early so as to keep me healthy.

2. IN ORDER THAT / SO THAT + S + V

  • Nam gets up early so that he has enough time for breakfast.
  • Nam always washes his hand in order that he is safe from coronavirus.

CONCLUSION

  • IN ORDER TO / SO AS TO + V: cụm từ chỉ kết quả
  • IN ORDER THAT / SO THAT + S + V: mệnh dề chỉ kết quả
